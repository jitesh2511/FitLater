# What is FitLater?
FitLater is a tool used to explore data and provide powerful insights which helps in building better ML models. It helps the user to better understand the data they are working with and helps them make better decisions regarding their model.  

The primary purpose of the tool is to provide clarity to the user regarding their data. After using the tool the users should have clarity of how their data is behaving.

It protects the user from leaving out any useful insights while rushing in building their models and provides complete information on how to mold their data for training.

However, this tool should not be confused with an AutoML as it does not perform hyperparameter tuning, suggests best algorithm with least loss, perform extensive cross-validations, etc. It only performs exploratory data analysis on behalf of the user.
