What is FitLater?

FitLater is an EDA tool which helps the user understand their data and make better decisions for their ML Model preprocessing.

  • Follows a structured EDA pipeline instead of scattered analysis
  • Goes beyond statistics to identify problems in your dataset
  • Provides clear, reasoned advice on what to fix and why
  • Built to bridge the gap between data understanding and preprocessing

What FitLater is NOT

❌

Not limited to generating charts and summary statistics

❌

Not a no-code dashboarding or BI solution

❌

Not focused on model training or predictions

❌

Not designed to replace human intuition, but to support it

Architecture

Descriptive

Generates foundational statistics and summaries to provide an initial understanding of the dataset structure and distributions

Diagnostics

Analyzes the dataset to identify data quality and statistical issues such as missing values, outliers, and weak relationships

Advisory

Translates detected issues into prioritized recommendations, helping guide preprocessing decisions with clear reasoning

Features

Missing Value Detection

Identifies and summarizes missing data across columns

Outlier Detection

Detects extreme values using statistical methods

Correlation Analysis

Finds relationships between numerical features

Distribution Analysis

Understands data spread and feature behavior

Priority-based Advisory

Categorizes issues into high, medium, and low priority

CLI + Frontend Support

Accessible via command line and interactive UI
